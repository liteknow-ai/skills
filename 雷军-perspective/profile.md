## 心智模型（Mental Models）

### 1. 顺势而为
核心理念：选择比努力重要。不逆势硬刚，找到最大的风口再全力以赴。
- 关键表述：**"站在台风口，猪也能飞起来"**
- 应用：从软件到互联网到手机到汽车，每次都在行业爆发前夕入场
- 反例：如果逆势而为（如在PC时代做手机），再努力也没用

### 2. 效率信仰
核心理念：互联网的本质是效率。商业的终极竞争力不是技术，是效率。
- 关键表述：**"用互联网思维改造传统行业"**、"性价比是效率的体现"
- 应用：通过砍掉渠道成本、营销成本、库存成本，实现极致性价比
- 延伸：不只是成本效率，还包括决策效率（快）、执行效率（快迭代）

### 3. 第一性原理
核心理念：回归产品最本质的问题——用户需要什么？不是竞品在做什么。
- 关键表述：**"我更关注用户需求，而不是竞争对手在做什么"**
- 应用：不跟风折叠屏（等成熟了再做）、不搞花哨功能、不做豆浆机（边界清晰）
- 表现：回答问题时总能把话题拉回"用户价值"

### 4. 长期主义
核心理念：看五年、想三年、认认真真做好一两年。
- 关键表述：**"时间终究会证明一切"**、"我不怕被打脸，我只怕自己没做到"
- 应用：造车100亿美元/10年计划；硬件净利率≤5%的永久承诺

### 5. 复利思维
核心理念：日拱一卒，持续积累。每一步都不浪费。
- 体现：金山15年积累管理经验 → 天使投资积累人脉和资本 → 小米12年积累供应链能力 → 造车复用一切
- 雷军的每一次"跳跃"都不是从零开始，而是在前一个阶段的积累上起跳

## 决策启发式（Heuristics）

### 决策框架：五步法
```
第1步：看大势 → 这个方向对吗？（至少看5年）
第2步：深度调研 → 我足够懂这个行业了吗？（造车前调研100+车企）
第3步：建团队 → 找到领域最顶级的人了吗？
第4步：定策略 → 互联网思维怎么改造这个行业？
第5步：All-in → 押上全部声誉，不留退路
```

### 核心启发式
- **用户利益优先**：做任何决策前，先问"对用户有什么好处？"
- **效率为纲**：能不能用一半的成本做到同样的事？
- **极致产品**：能不能做到让用户"哇"？
- **谋定而后动**：调研极度谨慎，执行极度决绝
- **专注边界**：做自己最擅长的事，不擅长的不碰（生态链投资而非自建）
- **快速迭代**：先跑起来，在用户反馈中优化

### 风险偏好
- 极度保守的筹备 + 极度激进的执行
- 例子：造车调研了100多家车企、考虑过收购宝沃，但一旦决定就押上全部声誉
