# Vosk 中文 ASR 常见错误模式参考

本文档记录了 Vosk small-cn 模型在中文语音识别中的常见错误模式，用于辅助人工校对。

## 一、Vosk 中文输出特征

### 1. 逐词空格
Vosk 中文输出在每个词之间都插入空格：
```
原始: 我们 每天 一 睁眼 就 在 和 钱 打交道
正确: 我们每天一睁眼就在和钱打交道
```
**处理方式**：用脚本自动去除中文字符间的空格，保留英文/数字周围的空格。

### 2. 无标点
Vosk small 模型不输出标点符号，需要后处理添加。
**处理方式**：根据语义停顿手动添加，或使用标点恢复模型。

### 3. 无段落分隔
全部文本输出为一整行，需要手动分段。
**处理方式**：在内容转折处（如"第一步""然而""但是"）插入段落。

## 三、错误识别规律

1. **同音字替换**：小模型倾向于选择最常见的同音字，而非上下文正确的字
2. **专业术语分解**：将专业术语拆分成常见词汇（如"通货膨胀"→"通过膨胀"）
3. **数字混淆**：相近发音的数字容易混淆（如"零"和"领"、"亿"和"一"）
4. **专有名词失败**：对人名、机构名、产品名识别率低
5. **口语化表达**：对口语中的语气词、连接词识别不稳定

## 四、校对优先级

1. **最高**：数字、金额、比率（直接影响信息准确性）
2. **高**：专有名词、人名、机构名
3. **中**：金融术语、专业概念
4. **低**：语气词、连接词（不影响理解）

## 五、扩展修正表

`scripts/cleanup_transcript.py` 中内置了 200+ 条常见错误修正映射。针对不同领域的视频内容，可能需要补充特定术语的修正。建议：

1. 先运行自动清理脚本
2. 通读一遍，标记不通顺的句子
3. 针对新发现的错误模式，添加到 ASR_FIXES 列表中
4. 重新运行清理脚本
